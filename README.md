## Quick Start

### Installation

```shell
# 1. л§Ҳмј“н”Ңл ҲмқҙмҠӨ л“ұлЎқ
/plugin marketplace add joyuno/pulse

# 2. н•өмӢ¬ м—”м§„ м„Өм№ҳ (мқҙкІғл§ҢмңјлЎң лҸҷмһ‘)
/plugin install pulse@pulse

# 3. лӘЁл“Ҳ м¶”к°Җ (м„ нғқ вҖ” мһҲмңјл©ҙ лҚ” к°•л Ҙ)
/plugin install pulse@pulse-interview    # мҶҢнҒ¬лқјн…ҢмҠӨ мқён„°л·°
/plugin install pulse@pulse-qa           # 3-Tier QA + Playwright
/plugin install pulse@pulse-contracts    # мқён„°нҺҳмқҙмҠӨ кі„м•Ҫ
/plugin install pulse@pulse-immunity     # мӢӨнҢЁ л©ҙм—ӯ мӢңмҠӨн…ң
/plugin install pulse@pulse-dna          # мӮ¬мҡ©мһҗ н”„лЎңнҢҢмқј н•ҷмҠө
```

### Direct Installation

```shell
# кёҖлЎңлІҢ мҠӨнӮ¬лЎң м§Ғм ‘ ліөмӮ¬
cp -r skills/* ~/.claude/skills/
```

### 1. Socratic Interview вҖ” мҶҢнҒ¬лқјн…ҢмҠӨ мқён„°л·°
